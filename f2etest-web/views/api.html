ApiKey: <%=apiKey%>

/version: 获得当前F2etest版本信息

  • DEMO地址：http://<%=host%>/version

/openapp: 使用F2etest中的浏览器打开指定URL：

  • DEMO地址：http://<%=host%>/openapp?id=ie8&url=http://www.alibaba.com/
  • 此接口支持以下参数：
    1. id: 浏览器ID，例如：ie6, ie7, ie8, ie9, ie10, ie11, chrome, firefox
    2. url: 需要打开的目标URL地址
    3. noproxy: 是否禁用代理，设置1为禁用，禁用后即访问线上
    4. proxyurl: 代理服务器pac地址，可利用代理服务器实现自定义hosts绑定功能
    5. params: program的参数，如果设置了此值，上方的3个参数将失效：url、noproxy、proxyurl
  • 注：此接口支持自动调用initUser，调此接口前无需提前调用initUser

/getAllBrowsers: 获取所有浏览器列表：

  • DEMO地址：http://<%=host%>/getAllBrowsers?callback=cbfunc
  • 此接口支持以下参数：
    1. callback: jsonp的回调函数(可省略)

/initUser: 为当前用户初始化账号：

  • DEMO地址：http://<%=host%>/initUser?callback=cbfunc
  • 此接口支持以下参数：
    1. callback: jsonp的回调函数(可省略)

/setHosts: 设置当前为hosts模式，并传入hosts绑定：

  • DEMO地址：http://<%=host%>/setHosts?apikey=<%=apiKey%>&name=<%=userid%>&hosts=<%=userIp%> www.alibaba.com&callback=cbfunc
  • 此接口支持以下参数(除callback参数以外，均兼容POST提交)：
    1. apikey: apikey
    2. name: 用户名(您的用户名为：<%=userid%>)
    3. hosts: hosts绑定
    4. callback: jsonp的回调函数(可省略)

/setHostsForward: 设置当前为反向代理模式，并配置反向代理参数：

  • DEMO地址：http://<%=host%>/setHostsForward?apikey=<%=apiKey%>&name=<%=userid%>&forwardHost=<%=userIp%>&forwardPort=8888&callback=cbfunc
  • 此接口支持以下参数(除callback参数以外，均兼容POST提交)：
    1. apikey: apikey
    2. name: 用户名(您的用户名为：<%=userid%>)
    3. forwardHost: 反向代理IP地址(如果值为：local, 则设置为访问者的IP)
    4. forwardPort: 反向代理端口号
    5. callback: jsonp的回调函数(可省略)

/getHostsConfig: 获取当前的hosts配置：

  • DEMO地址：http://<%=host%>/getHostsConfig?apikey=<%=apiKey%>&name=<%=userid%>&callback=cbfunc
  • 此接口支持以下参数(除callback参数以外，均兼容POST提交)：
    1. apikey: apikey
    2. name: 用户名(您的用户名为：<%=userid%>)
    3. callback: jsonp的回调函数(可省略)

/getHostsPac: 获取代理PAC：

  • DEMO地址：http://<%=host%>/getHostsPac?name=<%=userid%>
  • 此接口支持以下参数：
    1. name: 用户名(您的用户名为：<%=userid%>)
